# 👑 Lord'Os - نظام مروى الذكي

نظام تشغيل هجين (PWA) يعمل كموقع ويب وتطبيق جوال مع وكيل ذكي بشخصية نجدية أصيلة.

## 🤖 مروى مسلم الدوسري - الوكيل الذكي

**الهوية والشخصية:**
- وكيلة ذكية بلهجة نجدية بيضاء (مفهومة وقريبة للقلب)
- تستخدم مفردات نجدية مثل "سمّ"، "أبشر بعزك"، "يا بعد حيّي"
- تمزج بين الفزعة السعودية والدقة التقنية

**القدرات الوظيفية:**
- محللة ارتباطات سلوكية وأنماط حياتية
- مكتشفة أنماط صحية ومزاجية
- مديرة رفاهية شخصية

## 📁 ملفات النظام

### الملفات الرئيسية:
1. **index.html** - واجهة النظام (سطح المكتب + الجوال)
2. **style.css** - التصميم الملكي (Dark Mode + Glassmorphism)
3. **brain.js** - محرك مروى الذكي (منطق التفاعل والتحليل)
4. **manifest.json** - ملف تحويل الموقع لتطبيق جوال (PWA)

### التطبيقات المدمجة:
- 🤖 **مروى AI** - محادثة ذكية مع الوكيل
- 📊 **تحليل الحياة** - لوحة قيادة ورؤى ذكية
- 📝 **تسجيل العادات** - تتبع الأنماط اليومية

## 🚀 التشغيل المحلي

\`\`\`bash
# افتح الملف مباشرة في المتصفح
open index.html

# أو استخدم خادم HTTP محلي
python3 -m http.server 8080
# ثم افتح: http://localhost:8080
\`\`\`

## 🌐 النشر (Deployment)

### الخيار 1: Netlify (مجاني)
1. اذهب إلى [Netlify Drop](https://app.netlify.com/drop)
2. اسحب الملفات الأربعة وأفلتها
3. ستحصل على رابط مباشر للنظام

### الخيار 2: GitHub Pages
1. ارفع الملفات إلى repository
2. فعّل GitHub Pages من الإعدادات
3. سيكون النظام متاحاً على: `username.github.io/repo-name`

### الخيار 3: Vercel
\`\`\`bash
npm i -g vercel
vercel --prod
\`\`\`

## 📱 التثبيت كتطبيق جوال

1. افتح الرابط من جوالك
2. اضغط على قائمة المتصفح
3. اختر "Add to Home Screen" (إضافة للشاشة الرئيسية)
4. الآن أصبح لديك تطبيق Lord'Os بأيقونة خاصة

## 🎨 المميزات التقنية

- **PWA (Progressive Web App)** - يعمل بدون اتصال بالإنترنت
- **Dark Mode** - تصميم داكن يريح العين
- **Glassmorphism** - تأثيرات زجاجية عصرية
- **RTL Support** - دعم كامل للعربية
- **Mobile-First** - مُحسّن للجوال أولاً
- **Serverless** - لا يحتاج خادم backend

## 🔮 التطوير المستقبلي

- ربط مع API حقيقي (ChatGPT/Gemini)
- قاعدة بيانات محلية (IndexedDB)
- تحليل بيانات متقدم
- إشعارات ذكية
- مزامنة سحابية
